# graphql-getting-started

Studying GraphQL.

## Install

Simple run:

```sh
yarn
```

## Running

For development run:

```sh
yarn dev
```

Or for production:

```sh
yarn start
```

Then visit [localhost:3000/graphql](http://localhost:3000/graphql)
